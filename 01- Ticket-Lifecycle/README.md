Password Reset Lab

Objective:
Reset a user's password using the IT support ticket process.

Tools Used:
- ChatGPT lab
- Windows laptop

Steps Performed:
1. Received the password reset request.
2. Verified the user's identity.
3. Reset the password.
4. Asked the user to sign in with the temporary password.
5. Confirmed the issue was resolved.
6. Closed the ticket.

Outcome:
The user's password was successfully reset and access was restored.

Skills learned:
- Password reset procedures
- User verification
- Ticket documentation
- Customer service
