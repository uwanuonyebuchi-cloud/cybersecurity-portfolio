#  Wireshark Packet Analysis

## Overview

This lab demonstrates the use of **Wireshark**, a widely used network protocol analyzer, to capture and analyze network traffic. The objective was to inspect packets, identify common network protocols, understand network communication, and recognize normal versus suspicious traffic patterns.

---

## Objective

The objective of this lab was to:

- Capture and analyze network packets.
- Identify common network protocols.
- Examine packet details and communication flows.
- Understand how devices communicate across a network.
- Develop foundational packet analysis skills used in cybersecurity investigations.

---

## Lab Environment

- Wireshark
- Windows 11
- GitHub

---

## Scenario

As a cybersecurity analyst, you are tasked with analyzing network traffic to understand how systems communicate and identify any unusual or suspicious activity. Using Wireshark, packet captures were reviewed to examine protocols, source and destination addresses, and packet contents.

---

## Activities Performed

- Captured live network traffic.
- Opened and reviewed packet captures.
- Applied display filters to isolate specific protocols.
- Inspected packet headers and payloads.
- Identified source and destination IP addresses.
- Examined TCP sessions and network conversations.
- Documented findings from the packet analysis.

---
