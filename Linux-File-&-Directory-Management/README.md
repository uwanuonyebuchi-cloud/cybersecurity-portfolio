# Linux File & Directory Management

## Overview

This lab demonstrates fundamental Linux file and directory management using the command line. The objective was to create, navigate, organize, and manage directories and files within a Kali Linux environment. These are essential Linux administration skills commonly used in IT Support, System Administration, and Cybersecurity roles.

---

## Objective

The objective of this lab was to:

- Navigate the Linux file system.
- Create and organize directories.
- Create and manage files.
- Verify directory locations.
- Practice essential Linux command-line operations.

## Commands Used

Display current directory:

```bash
pwd
```

List files and directories:

```bash
ls
```

List detailed file information:

```bash
ls -la
```

Create a new directory:

```bash
mkdir cyberlab
```

Navigate into a directory:

```bash
cd cyberlab
```

Create a new file:

```bash
touch notes.txt
```

Return to the previous directory:

```bash
cd ..
```
