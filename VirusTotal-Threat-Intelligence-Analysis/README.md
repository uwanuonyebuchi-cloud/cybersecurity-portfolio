# VirusTotal Threat Intelligence Analysis

## Overview

This lab demonstrates the use of **VirusTotal**, a cloud-based threat intelligence platform, to investigate suspicious Indicators of Compromise (IOCs). The objective was to analyze URLs, domains, and file hashes using multiple antivirus engines and threat intelligence sources to determine whether they were malicious or safe.

---

## Objective

The objective of this lab was to:

- Investigate suspicious URLs, domains, and file hashes.
- Analyze threat intelligence from multiple security vendors.
- Identify malicious Indicators of Compromise (IOCs).
- Assess the reputation of suspicious artifacts.
- Develop practical threat intelligence and malware analysis skills.

---

## Lab Environment

- VirusTotal
- Web Browser
- Windows 11
- GitHub

---

## Scenario

As a SOC Analyst, you received several suspicious Indicators of Compromise (IOCs) from a security alert. Your task was to use VirusTotal to determine whether the indicators were malicious, suspicious, or benign by reviewing detection results from multiple security vendors.

---

## Activities Performed

- Accessed the VirusTotal platform.
- Investigated suspicious URLs.
- Analyzed domain reputation.
- Reviewed detection results from multiple antivirus vendors.
- Examined file hash intelligence.
- Evaluated community reputation and threat information.
- Documented investigation findings.

## Investigation Findings

The investigation revealed:

- One suspicious URL was detected as **malicious** by multiple antivirus vendors.
- A separate domain returned **no malicious detections** at the time of analysis.
- Threat intelligence results provided reputation information, detection history, and vendor classifications.
- VirusTotal enabled quick validation of suspicious indicators before further investigation.
