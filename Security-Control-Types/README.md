# Lab 25 – Security Control Types

## Overview

This lab demonstrates an understanding of the three primary types of security controls used to protect an organization's information systems. Security controls are categorized as Administrative, Technical, and Physical controls, each playing an important role in maintaining confidentiality, integrity, and availability (CIA).

---

## Objective

The objective of this lab was to identify common security controls and correctly classify them based on their control type.

## Security Control Types

### Administrative Controls
Administrative controls are policies, procedures, and processes designed to manage security risks.

Examples:

- Security Policies
- Employee Security Awareness Training
- Background Checks
- Incident Response Plan
- Acceptable Use Policy
- Access Control Policy
- Security Risk Assessments

---

### Technical Controls

Technical controls use technology to protect systems, networks, and data from unauthorized access or attacks.

Examples:

- Firewalls
- Multi-Factor Authentication (MFA)
- Antivirus Software
- Endpoint Detection and Response (EDR)
- Intrusion Detection System (IDS)
- Intrusion Prevention System (IPS)
- Encryption
- Virtual Private Network (VPN)
- Access Control Lists (ACLs)

---

### Physical Controls

Physical controls protect people, facilities, and equipment from unauthorized physical access or damage.

Examples:

- Security Guards
- CCTV Cameras
- Door Locks
- Badge Access Systems
- Fences
- Alarm Systems
- Server Room Locks
- Biometric Access Controls
