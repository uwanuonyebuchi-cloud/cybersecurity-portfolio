# Linux Command Line Fundamentals

## Overview

This lab introduces the Linux command-line interface (CLI) and essential Bash commands used for system administration and cybersecurity tasks. The objective was to become familiar with navigating the Linux terminal, identifying the current user, managing directories, and understanding the basic command-line environment.

---

## Objective

The objective of this lab was to:

- Learn the fundamentals of the Linux command line.
- Navigate the Linux file system.
- Identify the current logged-in user.
- Understand the Bash shell.
- Practice commonly used Linux commands.

---

## Scenario

As an entry-level cybersecurity analyst, you need to become comfortable working in the Linux terminal. In this lab, you practiced essential Bash commands used daily by Linux administrators, IT support professionals, and cybersecurity analysts.

## Activities Performed

- Opened the Kali Linux terminal.
- Identified the currently logged-in user.
- Started and interacted with the Bash shell.
- Displayed the current working directory.
- Listed files and directories.
- Verified command execution.
- Learned how Linux responds to invalid commands.

---

## Commands Used

Display the current user:

```bash
whoami
```

Launch the Bash shell:

```bash
bash
```

Display the current working directory:

```bash
pwd
```

List files and directories:

```bash
ls
```

List detailed directory contents:

```bash
ls -la
```
