#  Role-Based Access Control (RBAC)

## Overview

This lab demonstrates the implementation of **Role-Based Access Control (RBAC)**, a security model that restricts access to systems and resources based on a user's job responsibilities. RBAC helps organizations enforce the **Principle of Least Privilege (PoLP)** by ensuring users only have access to the resources necessary to perform their duties.

---

## Objective

The objective of this lab was to:

- Understand the concepts of Role-Based Access Control (RBAC).
- Assign permissions based on job roles.
- Apply the Principle of Least Privilege.
- Manage user access to organizational resources.
- Improve security through controlled access management.
