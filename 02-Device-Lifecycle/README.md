## Device Lifecycle Stages

### 1. Device Procurement
- Receive a new laptop or desktop
- Verify the device matches the purchase order
- Inspect for damage

### 2. Asset Registration
- Record device information
- Assign an asset tag
- Document:
  - Manufacturer
  - Model
  - Serial Number
  - Asset ID

### 3. Device Configuration
- Install Windows
- Install drivers
- Install required applications
- Configure system settings

### 4. Security Configuration
- Install antivirus software
- Configure VPN
- Enable Windows Defender
- Apply Windows Updates
- Enable BitLocker (where applicable)

### 5. User Deployment
- Create or assign the user's account
- Join the device to the company domain (if applicable)
- Verify email and network access
- Test printers and shared drives

### 6. Maintenance
- Install software updates
- Monitor device health
- Replace failing hardware when necessary
- Resolve user-reported issues

### 7. Retirement
- Back up important data
- Securely wipe the storage device
- Remove the device from inventory
- Dispose of or recycle the hardware according to company policy

## Lab Scenario

A new employee joins the company and requires a company laptop.

The IT Support technician:

1. Receives the laptop
2. Records the asset information
3. Applies an asset tag
4. Installs Windows and required software
5. Installs antivirus and VPN
6. Performs Windows updates
7. Creates the user's account
8. Verifies all applications are working
9. Documents the deployment
10. Delivers the laptop to the employee

Later in the device's lifecycle, the laptop is securely wiped, removed from inventory, and retired following company procedures.
